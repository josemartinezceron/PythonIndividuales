# Justificación del permiso para usuarios distintos en un blog 
 para ver las diferencias debes usar estos usuarios:
 jose   josepassword      del Staff
 daniela danielapassword  escritores

En este proyecto de blog, se han establecido dos grupos de usuarios: el grupo "Staff de Administradores" y el grupo "Escritores". Los usuarios de administración incluyen a Macmini y Jose, mientras que los escritores son Amanda y Daniela. 
 
La asignación de permisos diferentes a estos grupos de usuarios tiene como objetivo garantizar un adecuado control y gestión del blog, así como mantener la integridad y la calidad de los contenidos publicados. A continuación, se justifica la necesidad de estos permisos para cada grupo: 
 
1. **Grupo "Staff de Administradores"**: 
   - Macmini y Jose son usuarios con roles de administradores del sitio. Tienen acceso a todas las funcionalidades y características del blog, incluyendo la capacidad de editar, eliminar y publicar cualquier publicación o página. 
   - Estos usuarios tienen la responsabilidad de supervisar y administrar todas las actividades del blog, asegurando que el contenido cumpla con los estándares establecidos y manteniendo el buen funcionamiento del sitio. 
   - Los permisos otorgados a este grupo les permiten realizar cambios en la configuración del blog, administrar usuarios, moderar comentarios y realizar cualquier otra tarea necesaria para mantener el sitio en funcionamiento. 
 
2. **Grupo "Escritores"**: 
   - Amanda y Daniela son usuarios con el rol de escritores. Se les ha asignado este rol específico para que puedan crear y editar sus propias publicaciones en el blog. 
   - Estos usuarios tienen acceso limitado a ciertas áreas del blog y solo pueden editar, eliminar y publicar sus propias publicaciones. No tienen permisos para modificar o eliminar las publicaciones de otros escritores o realizar cambios en la configuración del sitio. 
   - La asignación de permisos restringidos a este grupo garantiza la seguridad y la privacidad de las publicaciones de cada escritor, evitando cualquier modificación no autorizada o malintencionada. 
 
En resumen, la asignación de permisos diferentes para los grupos "Staff de Administradores" y "Escritores" en este blog es fundamental para mantener un control adecuado sobre las acciones realizadas por cada tipo de usuario. Esto asegura que el sitio funcione correctamente, se mantenga el contenido de calidad y se respete la privacidad y la autoría de las publicaciones de los escritores.
